# array_utility
Provides some basic Array utilities for Ruby.

## Installation
### RubyGems
``` shell
gem install array_utility
```

### Bundler
Add this line to the application's Gemfile:
``` ruby
gem 'array_utility'
```

Then execute:
``` shell
bundle
```
